# เว็บจองห้องพักโรงแรม

สามารถเลือกวันจองห้องพักพร้อมคำนวณค่าใช้จ่ายตามวันที่เข้าพัก

## ภาษาที่ใช้ในการพัฒนาระบบ

Frontend
- HTML CSS Boostrap 
ใช้ทำ UX/UI หน้าเว็บ

Backend
- PHP 
จัดการระบบเชื่อมต่อคุยกับ Database

Database
- MySQL 
เก็บข้อมูลห้อง, ผู้ใช้
